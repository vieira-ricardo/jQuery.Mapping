jQuery.Mapping — Draw Maps for companies, easy tracking system

Created By

  • Ricardo Vieira - Plugin created for all community. All the code can be used in other projects. With this plugin you can for example, track all the vacations of members in a company easily, or expenses or whatever you need. It is responsive to the size of the screen. See the example for more information.

Example

Mapping Configuration 

	$("#demo").mapping({
		source: dataSource,
		holidays: dataHolidays,            
		year: 2013,
		multiSelect: false,            
		groupingView: {
			groupCollapse: false,
			sort: [{
				groups: ["name", "desc"],
				order: "desc"
			}]
		},
		group1Text: "Collaborator",
		group2Text: "Vacations",		
		headerYear: "Current Year:",
		headerTitle: "",
		errorText: "No Data",
		filter: ["filter", "graph"],            
		scale: "month",
		sumText: ["Day","Days"]
	});
Parameter Default Accepts Type
source null JSON
holidays null JSON
year current year Integer
group1Text "" String
group2Text "" String
headerTitle "" String
holidaysTitle "" String
filter ["filter", "graph", "facebook"] Array
multiSelect false Boolean
sumText ["Day","Days"] Array
scale "month" String
groupingView groupCollapse: false, sorting: desc Array
errorText "Source not available" String
waitText "Loading, Please Wait.." String
events Automatic Automatic
waitText "Please Wait..." String

Source Configuration 

   source: [{
	  id: 1,
	  desc_id: 1,
	  name: "David",
	  desc: "Sick",
	  values: [ ... ]
   }]
Parameter Default Accepts Type Meaning
id null Integer ID of first column
desc_id null Integer ID of second column
name null String Text that appears on first column
desc null String Text that appears on second column
values null Array Collection of dates for mapp items. See next table.

Value Configuration 

   values: [{
        date: "20/02/2013",
        customLabel: "D",
        customTitle: "David is Sick",
        customClass: "mapBlue",
        dateid: "1"
   }]
Parameter Default Accepts Type Meaning
date null String (Date) Date of occurrence. format DD-MM-YYYY or DD/MM/YYYY
customLabel null String Text that appears on date
customTitle null String Text that appears on title when mouse hover
customClass null String Class applied to occurrence
dateid null String ID of fate occurrence

Holidays Configuration 

   holidays:  [{

   }]

date null String (Date) Date of holiday. format DD-MM-YYYY or DD/MM/YYYY
title null String Text that appears on holiday
dow null String Text that appears on date of holiday
